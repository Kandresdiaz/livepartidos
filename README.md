# 🚀 Proyecto Express: Web Final del Mundial en Vivo

Este proyecto es una aplicación Web Front-End liviana, súper responsiva y con **diseño premium en modo oscuro (Glassmorphic UI)** para la cobertura en tiempo real de partidos de fútbol.

---

## ⚡ Características Principales

1. **Marcador & Minuto a Minuto 100% Automático**: Integra un widget en vivo que no requiere actualización manual.
2. **Predictor de IA Simulado en Vivo**: Muestra dinámicamente consejos de apuestas e incidencias que cambian cada 35 segundos para enganchar a los usuarios.
3. **Optimizado para Monetización**: Botón CTA de alta conversión para tus enlaces de Adsterra, PropellerAds o casas de apuestas (Betplay).
4. **Cero Dependencias**: HTML/CSS/JS nativo puro. Carga ultra rápida sin frameworks.

---

## 🛠️ Comandos Git Rápidos para Desplegar en GitHub Pages

Abre tu terminal en la carpeta del proyecto y ejecuta:

```bash
# 1. Inicializar repositorio Git local
git init

# 2. Agregar el archivo al staging y hacer commit
git add .
git commit -m "feat: landing de final en vivo lista"

# 3. Vincular con tu repositorio de GitHub (Remplaza tu-usuario)
git branch -M main
git remote add origin https://github.com/TU-USUARIO/final-mundial-live.git

# 4. Subir a GitHub
git push -u origin main
```

---

## 🌐 Publicar en GitHub Pages en 3 Clics

1. Entra a tu repositorio en GitHub: `https://github.com/TU-USUARIO/final-mundial-live`.
2. Ve a **Settings > Pages**.
3. En **Branch**, selecciona `main` y guarda.
4. Tu web quedará pública de inmediato en `https://tu-usuario.github.io/final-mundial-live/`.
